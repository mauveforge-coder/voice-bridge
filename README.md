# VoiceBridge

iPhoneの高精度音声認識テキストを、Windows PCのフォーカス中ウィンドウにそのまま流し込むツール。

![demo](demo.gif)

---

## 特徴

- iPhoneのネイティブ音声認識（高精度・日本語対応）をそのまま利用
- ブラウザページを開くだけ。専用アプリ不要
- Python不要のスタンドアロンexe版あり
- SendInput API経由で日本語を確実に注入（クリップボード経由）

---

## 必要環境

| 項目 | 要件 |
|------|------|
| PC | Windows 10 / 11 |
| スマートフォン | iPhone（Safari） |
| ネットワーク | iPhoneとPCが同一WiFiに接続されていること |

---

## セットアップ手順

### exe版（推奨・Python不要）

1. [Releases](https://github.com/mauveforge-coder/voice-bridge/releases) から `voice-bridge.exe` をダウンロード
2. `voice-bridge.exe` をダブルクリックして起動
3. ターミナルにQRコードが表示される → iPhoneでスキャン

### ソースから実行（開発者向け）

```bash
git clone https://github.com/mauveforge-coder/voice-bridge.git
cd voice-bridge
pip install qrcode
python server.py
```

---

## 使い方

1. QRコードをiPhoneのカメラでスキャンして Safari でページを開く
2. テキストエリアをタップ → キーボードのマイクボタンで話す
3. 「PCに送信」ボタンをタップ
4. 0.3秒後、PC上のフォーカス中ウィンドウ（ChatGPT・メモ帳・VSCode等）にテキストが貼り付けられる

## iOSショートカット設定手順（任意）

ホーム画面のボタン一発で音声認識→送信まで自動化したい場合は、2つのショートカットを使う。

---

### セットアップ用ショートカット「ボイスブリッジセットアップ」

サーバーを再起動するたびに1回だけ実行する。QRコードをスキャンしてURLとトークンをiCloudに保存する。

| # | アクション | 設定 |
|---|-----------|------|
| 1 | **QRまたはバーコードをスキャン** | （設定なし） |
| 2 | **テキストを分割** | 入力: QR/バーコード / セパレータ: カスタム `?t=` |
| 3 | **テキストを結合** | 入力: テキストを分割 / セパレータ: カスタム `input/?t=` |
| 4 | **ファイルを保存** | 入力: 結合済みのテキスト / 保存先を尋ねる: オフ / サブパス: `/Shortcuts/vb_token.txt` / ファイルが存在する場合は上書き: オン |

> ステップ3の結合により `http://PCのIP:9876/input/?t=TOKEN` という形式で1ファイルに保存される。

---

### 送信用ショートカット「ボイスブリッジ」

毎回使うショートカット。

| # | アクション | 設定 |
|---|-----------|------|
| 1 | **フォルダからファイルを取得** | パス: `vb_token.txt` / フォルダ: `Shortcuts` |
| 2 | **テキストを分割** | 入力: ファイル / セパレータ: カスタム `/?t=` |
| 3 | **リストから項目を取得** | テキストを分割 から **最初の項目**（= URL） |
| 4 | **変数に追加** | 変数名: `inputURL` |
| 5 | **リストから項目を取得** | テキストを分割 から **最後の項目**（= トークン） |
| 6 | **変数に追加** | 変数名: `token` |
| 7 | **テキストを音声入力** | （設定なし） |
| 8 | **URLの内容を取得** | 下記参照 |

「URLの内容を取得」の詳細設定：

| 項目 | 値 |
|------|----|
| URL | 変数 `inputURL` |
| 方法 | POST |
| ヘッダー キー | `X-VoiceBridge-Token` |
| ヘッダー 値 | 変数 `token` |
| 本文を要求 | JSON |
| フィールド キー | `text` |
| フィールド 値 | 音声入力されたテキスト |

---

### ホーム画面に追加する（任意）

「ボイスブリッジ」ショートカットを長押し → 「詳細」→「ホーム画面に追加」

## トラブルシューティング

### テキストが入力されない

- `http://PCのIP:9876/ping` にブラウザでアクセスして `{"status":"alive"}` が返るか確認する
- iPhoneとPCが同じWiFiに接続されているか確認する
- WindowsファイアウォールがポートをブロックしていないかPC側で確認する

### ファイアウォールの設定（必要な場合）

```
Windowsセキュリティ → ファイアウォールとネットワーク保護
→ 詳細設定 → 受信の規則 → 新しい規則
→ ポート → TCP → 9876 → 接続を許可する
```

### IPアドレスが変わった場合

自宅ルーターでPCに固定IPを割り当てるか、ショートカットのURLを都度更新する。

## 技術仕様

| 項目 | 内容 |
|------|------|
| 使用言語 | Python 3（標準ライブラリのみ） |
| HTTPサーバー | `http.server.HTTPServer` |
| ポート | 9876 |
| 文字注入方式 | Windows SendInput API（ctypes・クリップボード経由） |
| 対応文字範囲 | Unicode BMP（U+0000〜U+FFFF） |
| 絵文字（サロゲートペア） | スキップ |
| 注入遅延 | 0.3秒 |
