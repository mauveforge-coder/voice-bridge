# VoiceBridge

iPhoneの高精度音声認識テキストを、Windows PCのフォーカス中ウィンドウにそのまま流し込むツール。

![demo](demo.gif)

---

## 特徴

- iPhoneのネイティブ音声認識（高精度・日本語対応）をそのまま利用
- ブラウザページを開くだけ。専用アプリ不要
- Python不要のスタンドアロンexe版あり
- SendInput API経由で日本語を確実に注入（クリップボード経由）

---

## 必要環境

| 項目 | 要件 |
|------|------|
| PC | Windows 10 / 11 |
| スマートフォン | iPhone（Safari） |
| ネットワーク | iPhoneとPCが同一WiFiに接続されていること |

---

## セットアップ手順

### exe版（推奨・Python不要）

1. [Releases](https://github.com/mauveforge-coder/voice-bridge/releases) から `voice-bridge.exe` をダウンロード
2. `voice-bridge.exe` をダブルクリックして起動
3. ターミナルにQRコードが表示される → iPhoneでスキャン

### ソースから実行（開発者向け）

```bash
git clone https://github.com/mauveforge-coder/voice-bridge.git
cd voice-bridge
pip install qrcode
python server.py
```

---

## 使い方

1. QRコードをiPhoneのカメラでスキャンして Safari でページを開く
2. テキストエリアをタップ → キーボードのマイクボタンで話す
3. 「PCに送信」ボタンをタップ
4. 0.5秒後、PC上のフォーカス中ウィンドウ（ChatGPT・メモ帳・VSCode等）にテキストが貼り付けられる

---

## iOSショートカット設定手順（任意）

ホーム画面のボタン一発で音声認識→送信まで自動化したい場合は、iOSショートカットを使う。

### ステップ 1: ショートカットを新規作成する

1. iPhoneで「ショートカット」アプリを開く
2. 右上「+」をタップ → 名前を入力（例: `ボイスブリッジ`）

### ステップ 2:「テキストを入力」アクションを追加する

1. 「アクションを追加」→ 検索欄に「テキストを入力」と入力
2. アクションを追加して「音声入力を使用」をオンにする

### ステップ 3:「URLの内容を取得」アクションを追加・設定する

1. 「アクションを追加」→ 「URLの内容を取得」を追加
2. 以下の通り設定する

| 項目 | 設定値 |
|------|--------|
| URL | `http://PCのIPアドレス:9876/input` |
| メソッド | POST |
| リクエストの本文 | JSON |

3. 「新しいフィールドを追加」→ キー: `text`、値: 「テキストを入力の結果」を選択

### ステップ 4: ホーム画面に追加する（任意）

ショートカットを長押し → 「詳細」→「ホーム画面に追加」

## トラブルシューティング

### テキストが入力されない

- `http://PCのIP:9876/ping` にブラウザでアクセスして `{"status":"alive"}` が返るか確認する
- iPhoneとPCが同じWiFiに接続されているか確認する
- WindowsファイアウォールがポートをブロックしていないかPC側で確認する

### ファイアウォールの設定（必要な場合）

```
Windowsセキュリティ → ファイアウォールとネットワーク保護
→ 詳細設定 → 受信の規則 → 新しい規則
→ ポート → TCP → 9876 → 接続を許可する
```

### IPアドレスが変わった場合

自宅ルーターでPCに固定IPを割り当てるか、ショートカットのURLを都度更新する。

## 技術仕様

| 項目 | 内容 |
|------|------|
| 使用言語 | Python 3（標準ライブラリのみ） |
| HTTPサーバー | `http.server.HTTPServer` |
| ポート | 9876 |
| 文字注入方式 | Windows SendInput API（ctypes・クリップボード経由） |
| 対応文字範囲 | Unicode BMP（U+0000〜U+FFFF） |
| 絵文字（サロゲートペア） | スキップ |
| 注入遅延 | 0.5秒 |
